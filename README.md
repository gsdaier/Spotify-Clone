# Spotify Clone (API) using Angular 
This is result of a tutorial in YTB, API Spotify using Angular, TS, SCSS, JS...

# Fully functional

## For execute the app
Use that command in terminal:
```sh
npm start
```

## For tutorial:
```sh
npm i @angular/cli -g
```
